## 引用关系说明

> **读法：** 矩阵第 R 行第 C 列 = 1，表示 **文件C 引用了 文件R**（C依赖R）

### index.html (F01) — 根节点
- 通过 `<script src>` 引用所有 F02~F21（共20个文件）
- 自身不被任何文件引用

### 二级节点引用的三级节点

| 二级节点 | 引用的三级节点 |
|----------|----------------|
| `entities.js` (F07)   | `items.js` (F16), `skills.js` (F18), `npcs.js` (F19) |
| `combat.js` (F08)     | `skills.js` (F18) — 读取SKILLS常量 |
| `building.js` (F09)   | `buildings.js` (F20) — 读取BUILDING_TEMPLATES |
| `resource.js` (F10)   | `items.js` (F16), `biomes.js` (F21) — 读取资源定义 |
| `craft.js` (F11)      | `items.js` (F16), `recipes.js` (F17) — 读取配方 |
| `cultivation.js` (F12)| `skills.js` (F18) — 读取SKILLS/REALMS |
| `social.js` (F13)     | `items.js` (F16), `npcs.js` (F19) — 读取物品价格和NPC模板 |
| `ui.js` (F15)         | `items.js` (F16), `recipes.js` (F17), `skills.js` (F18), `buildings.js` (F20) — 渲染面板 |
| `world.js` (F05)      | `biomes.js` (F21) — 调用getBiomeAt/getBaseSpiritAt |

## 数值联动关系图

```
灵力浓度(biomes.js) ──→ 修炼速度(cultivation.js)
                    ──→ 采集速度(resource.js)
                    ──→ HP恢复速度(entities.js)

天气系统(weather.js) ──→ 灵力倍率 ──→ 修炼速度
                    ──→ 移动速度(entities.js)
                    ──→ 篝火消耗速度(building.js)
                    ──→ 免疫力变化(entities.js)

饥饿值(entities.js) ──→ 移动速度
                   ──→ 免疫力下降
                   ──→ HP持续减少

免疫力(entities.js) ──→ HP恢复效率
                   ──→ 修炼效率
                   ──→ 生病概率(吃生肉)

生病状态(entities.js) ──→ HP持续减少
                     ──→ 修炼效率降低(-70%)
                     ──→ 免疫力持续下降

篝火(building.js) ──→ 烹饪解锁(craft.js)
                 ──→ 靠近可烤肉(resource.js)
                 ──→ 抵消寒冷免疫损失(entities.js)

床(building.js) ──→ HP恢复(entities.js)
               ──→ 灵力恢复(entities.js)
               ──→ 饥饿恢复(entities.js)
               ──→ 免疫力恢复(entities.js)
               ──→ 体力恢复(entities.js)

木桩(building.js) ──→ 体力消耗(entities.js)
                  ──→ 经验获取(entities.js)

门(building.js) ──→ 物理碰撞体动态增删(physics.js)

境界(cultivation.js) ──→ hpMax上限
                    ──→ spiritMax上限
                    ──→ 解锁高阶建筑(building.js)

门派(cultivation.js) ──→ 功法修炼速度×1.3
                    ──→ 解锁专属功法(SKILLS)

战斗(combat.js) ──→ 玩家HP减少
               ──→ 灵力消耗(灵剑)
               ──→ 击败敌人获得XP/掉落物

合成(craft.js) ──→ 消耗灵力(alchemy)
              ──→ 消耗背包材料(entities.js)
              ──→ 生成新物品进背包

聚灵阵(building.js) ──→ 周围灵力浓度+0.5(world.js)
```
